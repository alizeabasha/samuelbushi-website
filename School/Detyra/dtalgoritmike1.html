OPERATORET BITWISE

Identifikoni operatoret bitwise qe jane perdorur ne kodin e meposhtem. Kryeni veprimet me operatoret bitwise.

#include
using namespace std;
int main()
{
int x = 19;
cout<<"x << 1 = “ << (x << 1);
cout<<"x >> 1 = “ << (x >> 1);
return 0;
}

Zgjidhje e ushtrimit:

Operatoret e perdorur bitwise ne kodin e mesiperm jane:

<< (left shift)
>> (right shift)


Outputi/rezultati ne kodin e mesiperm do te ishte:

x = 19 (10011)

x << 1 eshte: 38 ose (100110)

x >> 1 eshte: 9 ose (1001)
