OPERATORET LOGJIKE

Si eshte vlera e shprehjeve qe vijojne si te plote dhe si vlere booleane?

a  3
b  5 + 7
c  3 + (3 < 4)
d  (2 > 1) + (3 < 2)


Zgjidhje e ushtrimit:

a  3   eshte 3
b  5 + 7   eshte 12
c  3 + (3 < 4)    eshte 3 + false
d  (2 > 1) + (3 < 2)   eshte true + false
